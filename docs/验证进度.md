`from jittor_geometric.nn import GCNConv` 在**无 NVIDIA GPU** 的本机上，原先会因：

1. `edgesoftmax_op.cc` 无条件 `#include <cuda.h>` → 已打补丁（`#ifdef JIT_cuda`）
2. `cusparse_ops` 不存在（CPU 版 Jittor）→ 已改 `spmmcsr.py` / `spmmcoo.py` 可导入
3. `getweight` 等 CUDA 算子在 import 时编译 → 已改 `ops/__init__.py` 在 CPU 下跳过
4. `nn/conv/__init__.py` 会拉取 **einops**、**sympy** 等 → 需 `pip install einops sympy`
