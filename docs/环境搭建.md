# 本地 Jittor 环境搭建（Windows）

## 成功标准

```text
D:\miniconda3\envs\jittorgeometric\python.exe -c "import jittor as jt; print(jt.__version__); print(jt.has_cuda)"
```

能打印版本号且无 `ModuleNotFoundError` 即通过。

## 为何不用系统 Python 3.13

- `pip install jittor` 会拉旧版 NumPy 并尝试源码编译
- 本机未安装 MSVC/gcc，编译失败
- 官方与 JittorGeometric 推荐 **Python 3.10**

## 安装步骤（ASCII 路径，避免用户名含中文）

Miniconda **不能**装到 `C:\Users\范双双\...`，请使用 `D:\miniconda3`：

```powershell
# 1. 若尚未安装 Miniconda（仅首次）
$installer = "$env:USERPROFILE\Miniconda3-installer.exe"
Invoke-WebRequest -Uri "https://repo.anaconda.com/miniconda/Miniconda3-latest-Windows-x86_64.exe" -OutFile $installer
Start-Process -Wait $installer -ArgumentList "/InstallationType=JustMe","/RegisterPython=0","/S","/D=D:\miniconda3"

# 2. 创建环境
D:\miniconda3\Scripts\conda.exe create -n jittorgeometric python=3.10 -y

# 3. 安装 Jittor（CPU）
D:\miniconda3\envs\jittorgeometric\python.exe -m pip install -U pip
D:\miniconda3\envs\jittorgeometric\python.exe -m pip install jittor

# 4. 设置 Jittor 缓存到纯 ASCII 路径（重要）
setx JITTOR_HOME "D:\jittor_cache"

# 5. 安装 Jittor（Windows 已验证：1.3.7.16）
D:\miniconda3\envs\jittorgeometric\python.exe -m pip install "numpy==1.26.4" "jittor==1.3.7.16"

# 6. 验证（首次 import 会下载 MSVC 并编译，约 2 分钟）
D:\miniconda3\envs\jittorgeometric\python.exe -c "import os; os.environ['JITTOR_HOME']=r'D:\jittor_cache'; import jittor as jt; print(jt.__version__, jt.has_cuda)"
```

**当前本机验证结果（2026-06-01）**：`1.3.7.16`，`has_cuda=False`，CPU 张量运算正常。

**无 NVIDIA GPU 时**：已对本地 `JittorGeometric` 做 CPU 适配（跳过 CUDA 算子导入、`GCNConv(spmm=False)`）。验证：

```powershell
$env:JITTOR_HOME = "D:\jittor_cache"
D:\miniconda3\envs\jittorgeometric\python.exe -c "import jittor as jt; jt.flags.use_cuda=0; from jittor_geometric.nn import GCNConv; print('GCNConv OK')"
```

也可一键执行：`powershell -ExecutionPolicy Bypass -File scripts\setup_jittor_env.ps1`

## 常见问题

### 用户名含中文导致 Miniconda / Jittor 缓存失败

- Miniconda 安装到 `D:\miniconda3`（不要 `C:\Users\范双双\...`）
- 设置 `JITTOR_HOME=D:\jittor_cache`（不要默认 `C:\Users\范双双\.cache\jittor`）

### `import jittor` 时 cl.exe 报 C2440

**1.3.10+** 在 Windows 上易失败，请固定 **1.3.7.16** 并设置 `JITTOR_HOME`。仍失败时用 **WSL2 + conda** 或头歌在线环境。
