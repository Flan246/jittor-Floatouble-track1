# 第六届计图 · 热身赛 1（赛道一热身）

来源：[头歌 Jittor-7 · 赛题说明](https://www.educoder.net/competitions/Jittor-7)

## 赛题名称

**计图挑战赛赛道一热身赛** — 基于 Cora 引文网络的 GCN 节点分类

## 任务

- 使用 **Jittor + JittorGeometric**
- 数据：`data/cora.pkl`（发布包内）
- 模型：**GCN** 节点分类
- 输出：`result.json`（测试集节点 id → 类别 0–6）

## 数据规模

| 项 | 值 |
|----|-----|
| 节点 | 2708 |
| 边 | 5429 |
| 特征维 | 1433 |
| 类别 | 7 |

字段：`x`, `y`, `edge_index`, `train_mask`, `val_mask`, `test_mask`；测试集 `y` 为 -1。

## 通过标准

- 测试集 **Accuracy ≥ 0.70**
- 每天最多提交 **2 次**
- 须通过此热身赛才能参加 **赛道一正式赛**（图学习动态推荐）

## 提交格式

```text
result.zip
  gcn.py        # 模型 + 训练代码
  result.json   # {"1708": 0, "1709": 1, ...}  key 为字符串
```
